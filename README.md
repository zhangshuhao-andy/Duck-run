CyberDuck is a neon cyberpunk parkour game developed based on Pygame, with exquisite visual effects, complete game systems and fluent operating experience.
Creator: Zhang Shuhao
Contact: 17868782889
This game provides ten progressive difficulty levels. New players can get intuitive operation guidance in the first level. 
Press Space to jump and double jump, and hold the C key to crouch and slide to avoid low obstacles. Various dangerous traps,
including mines, blocking lasers and swinging power grids, are distributed in each scene, testing players’ reaction and rhythm control.
Players can obtain coin rewards after clearing each level. The in-game shop supports skin unlocking. Multiple neon color styles such as cyan,
pink and purple are available for players to customize their duck. The character is designed with smooth squash deformation and ghost trail effects, 
matching the dynamic neon portal to build a strong futuristic atmosphere.
The game interface includes main menu, level selection, skin shop and about pages. 
All buttons have animated scaling feedback to enhance interaction. Optimized code ensures stable 60FPS operation.
With simple rules and high playability, it combines casual entertainment and skill challenges.
It is a lightweight and well-made independent parkour work.
